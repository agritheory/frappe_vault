## Frappe Vault

OpenBao integration for Frappe

### What is OpenBao?

[OpenBao](https://openbao.org) is an open-source fork of HashiCorp Vault, created after HashiCorp changed Vault's license from MPL-2.0 to BSL in 2023. OpenBao provides identity-based secrets and encryption management with enterprise-grade security features.

- **License**: MPL-2.0 (Mozilla Public License 2.0) - fully open source
- **Governance**: Governed by the [Open Source Security Foundation (OpenSSF)](https://openssf.org/)
- **API Compatibility**: Maintains API compatibility with Vault OSS v1.14.x

This integration is designed to meet compliance requirements for DCAA, CMMC, HIPAA, and SOC2 by providing cryptographic isolation of secrets, immutable audit trails, and per-customer isolation.
